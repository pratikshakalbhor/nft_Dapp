# FreelancerShield - NFT Project

This project contains the NFT features for the FreelancerShield platform.

## Structure
- `src/`: React frontend code.
- `backend/`: Node.js backend server.
- `contract/`: Stellar smart contracts.

## Setup & Running
1. **Frontend**:
   ```bash
   npm install
   npm start
   ```
   Runs on [http://localhost:3001](http://localhost:3001)

2. **Backend**:
   ```bash
   cd backend
   npm install
   npm start
   ```
   Runs on [http://localhost:5000](http://localhost:5000)

3. **Smart Contracts**:
   Located in the `contract/` directory.
